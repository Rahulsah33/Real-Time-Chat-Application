RR-Chat Application is a simple and responsive real-time chat interface that allows users to enter their name and send messages. The UI is clean, minimal, and designed for ease of use.


Features

 Real-time chat interface

 User name input before sending messages

 Clean and responsive UI

 Easy to integrate with WebSocket / Socket.IO / Spring Boot WebSocket backend

 Simple and beginner-friendly codebase


 Frontend:

HTML5

CSS3

Backend:

Spring Boot (WebSocket)
